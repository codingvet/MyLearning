# 命令行

win + R ->cmd                         打开命令窗口

d:                                                    切换到D盘

cd     文件名1\文件名2	 		   切换多级文件

table键										 补全命令

cd   ..                                              回到上级文件件

cd\                                                  回到根路径

dir                                                    查看文件夹

cls                                                    清屏

exit                                                   退出
